# Cradle Fusion 360 Add-in (Beta)

Welcome to the beta testing repository for the Cradle Fusion 360 Add-in! 

This repository is used to distribute the latest beta versions of the add-in and collect feedback, bug reports, and feature requests.

## Requirements

To use this add-in, you must have an active **whatt.io** account. 

1. **Register:** You can register for a free account at [whattio.app](https://whattio.app/) (a free account is sufficient to start, though you may upgrade to add more products).
2. **App Store:** Activate the integration by visiting the Cradle app page in the whatt.io App Store: [Cradle App Link](https://whattio.app/app/78dc7cc7-495f-4d42-a100-90638dc2dcd3)


## How to Install the Beta Add-in

To install the add-in manually from this repository, follow these steps:

1. Click the green **Code** button at the top right of this page and select **Download ZIP**.
2. Extract the downloaded ZIP file to a folder on your computer (e.g., your Documents folder). The extracted folder should contain `Cradle.py`, `Cradle.manifest`, and a `resources` folder.
3. Open Autodesk Fusion 360.
4. Press `Shift + S` on your keyboard to open the **Scripts and Add-Ins** dialog.
5. Click on the **Add-Ins** tab at the top of the dialog.
6. Click the green **"+"** icon next to "My Add-Ins".
7. Navigate to and select the folder you extracted in Step 2.
8. The add-in should now appear in your list. Select it and click **Run** to start it.
9. (Optional) Check the **Run on Startup** box if you want the add-in to start automatically when you open Fusion 360.

## How to Report Bugs or Request Features

We rely on your feedback to improve the app! If you encounter any issues or have ideas for new features:

1. Go to the **Issues** tab at the top of this repository.
2. Click **New Issue**.
3. Provide as much detail as possible, including screenshots if applicable, and steps to reproduce the bug.

Thank you for helping us test and improve the Cradle app!
